# BankManager

Application Java de gestion de comptes bancaires.

## Description
Ce projet implémente une hiérarchie de classes bancaires en Java
utilisant le concept d'héritage.

## Structure des classes
- `BankAccount` — classe parent (account, balance)
- `CheckingAccount` — compte courant (+ limit)
- `SavingsAccount` — compte d'épargne (+ interestRate)
- `COD` — certificat de dépôt (+ maturityDate)

## Concepts utilisés
- Héritage (extends)
- Encapsulation (getters/setters)
- Surcharge de toString()

## Comment exécuter
1. Cloner le projet
2. Ouvrir dans IntelliJ IDEA
3. Exécuter Main.java
